Provide your rating on your recent ice cream experience

Javascript Loops and Functions

A web page can be a great way of getting feedback from users. Here, we are asking the user for their name and a rating on a recent Ice Cream tasting experience they have had. After inputting their name, the user gives how many stars out of 5 they would like to give. This is then recorded and added to a list of previous results.

Note: The user should only be able to input a number between 0 to 5, with an error being displayed if it is outside these bounds
